# 产品需求文档 PRD

## 1. 产品名称
签证公司 Agent

## 2. 产品定位
面向签证公司、移民顾问团队和签证运营团队的企业级业务 Agent 平台。

## 3. 产品目标
通过标准化受理流程、材料管理、审核协同、进度跟踪和 AI 辅助能力，提升签证业务处理效率、降低漏项率、强化审计能力，并支持人工接管高风险环节。

## 4. 核心用户角色
- 创始人 / 管理员
- 客户经理
- 材料专员
- 审核员
- 财务或订阅管理员
- 客户 / 申请人

## 5. MVP 范围
- 租户管理与账号角色
- 客户与案件受理
- 材料清单与上传跟踪
- 案件流程状态管理
- 内部审核与审批队列
- 基础规则提醒
- 审计日志
- 仪表盘和 SLA 可见性

## 6. 暂不纳入 MVP
- 自动出具法律性质结论
- 完全无人审核通过
- 复杂 BI 平台
- 面向外部合作伙伴的开放市场

## 7. 成功指标
- 材料漏交率下降
- 案件处理周期缩短
- 审核返工率下降
- 高风险案件人工接管率可追踪
- 关键流程日志完整率达到目标阈值

## 8. 关键约束
- 必须支持人工介入
- 必须支持多租户隔离
- 必须支持审计
- 必须优先考虑 PII 与材料安全
