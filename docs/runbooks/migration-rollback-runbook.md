# Migration Rollback Runbook

## Purpose
Phase 1 rollback runbook skeleton for future L1/L2/L3 migrations.

## Minimum Steps
1. Identify migration classification and impact.
2. Stop follow-up migrations.
3. Review rollback plan under infra/db/rollback-plans.
4. Execute rollback steps.
5. Run validation queries.
6. Record result and incident notes.
