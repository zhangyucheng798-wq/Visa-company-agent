# 核心业务流程

## 1. 案件受理流程
1. 创建客户与申请人档案
2. 新建签证案件
3. 选择签证类型和受理模板
4. 生成材料清单
5. 分配客户经理或材料专员

## 2. 材料收集流程
1. 客户上传材料
2. 系统记录版本
3. AI 或规则引擎做初步校验
4. 材料专员复核
5. 缺失项或异常项进入待处理队列

## 3. 审核与审批流程
1. 案件进入审核队列
2. 审核员检查材料完整性与风险
3. 高风险案件升级人工审批
4. 审批结论形成可追踪记录

## 4. 进度与通知流程
1. 状态变更后记录审计日志
2. 根据规则触发内部或客户通知
3. SLA 风险案件进入仪表盘告警
